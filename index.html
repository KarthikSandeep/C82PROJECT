Project on console

A standalone source code editor that runs on Windows, macOS, and Linux. The top pick for JavaScript and web developers, with tons of extensions to support.

In JavaScript, the console is an object which provides access to the browser debugging A Web console is a tool which is mainly used to log information associated with a web page like: network requests, JavaScript, security errors, warnings, CSS etc. It enables us to interact with a web page by executing JavaScript expression in the contents of the page. Press f12 to open console To print something on the console use: console.log(); function
